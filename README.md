## ✨ Features

### Core Features
- ✅ **Real-time price tracking** - Live cryptocurrency prices with auto-refresh
- ✅ **Multi-currency support** - Track Bitcoin, Ethereum, and 10,000+ other cryptocurrencies
- ✅ **24-hour change percentage** - See price movements with bullish/bearish indicators
- ✅ **Interactive menu system** - User-friendly CLI interface
- ✅ **Search functionality** - Find any cryptocurrency by name or symbol
- ✅ **Price history** - Automatic tracking of last 100 price points
- ✅ **Error handling** - Graceful handling of API failures and network issues

### 🌟 Bonus Features
- 📊 **Price trend graphs** - Visualize price movements using matplotlib
- 💾 **CSV export** - Save price data to CSV files for analysis
- 🔔 **Alert system** - Set price threshold alerts (bonus feature)
- 🎨 **Color-coded output** - Visual indicators for price changes
- 📱 **Cross-platform** - Works on Windows, macOS, and Linux

## 📦 Prerequisites

Before you begin, ensure you have the following installed:

- **Python 3.7 or higher** - [Download Python](https://www.python.org/downloads/)
- **pip** - Python package manager (comes with Python)
